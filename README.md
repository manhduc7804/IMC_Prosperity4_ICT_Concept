# IMC Prosperity 4 - ICT Concept

Local workspace for IMC Prosperity strategies, organized by round.

## Structure

- `ROUND<round_number>/`: strategy, local backtest, datamodel, and data for each round

## Round details
- See `ROUND<round_number>/README.md` for round-specific details.

## Submission

- Edit `ROUND<round_number>/prosperity_submission.py`.
- Submit only that file to Prosperity (with required filename).

### Test locally
```bash
cd ROUND<round_number>
python backtest_from_csv.py
```
